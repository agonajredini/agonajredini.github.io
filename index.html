Hey, I’m Agon. I’m a programmer who enjoys building cool stuff with code. I am always trying to challenge myself and learn something new. Below are some of the fun personal projects I've worked on. If you want to reach out feel free to follow me on one of the links below.

NostalgjiTV

This project simulates a TV-like interface with YouTube videos. The user can control the video playback using keyboard commands (e.g., changing channels, adjusting volume, muting sound effects), and interact with a basic on-screen interface.

Treadmill To Strava

Treadmill to Strava is a Python-based application that allows users to upload treadmill workout data to Strava by extracting relevant information from images of treadmill screens. This project includes two graphical user interfaces (GUIs) implemented using Tkinter and Kivy for better accessibility and functionality.

Passing Network

Visualizing a football team's passing network by processing match event data to plot player positions, passing connections, and passing frequency on a football pitch
